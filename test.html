Finding #01

Missing Content-Security-Policy

The Content-Security-Policy (CSP) header is a powerful security layer that helps detect and mitigate certain types of attacks, including XSS.

Impact

Allows malicious script execution in the context of the user's browser session.

Recommendation

Implement a strict Content-Security-Policy that restricts script-src.
