# **App Name**: Pentellia

## Core Features:

- Top Horizontal Bar: A fixed black top bar with the Pentellia logo text on the left and space for profile/notifications on the right.
- Left Vertical Sidebar Navigation: A vertical sidebar with sections for workspace selection (simple dropdown placeholder) and links to Dashboard, Assets, Scans, Findings, Attack Surface, Handlers, Reports, Robots, Team, Integrations, and Settings.
- Main Content Area: A main content area with a neutral dark background and subtle card shadows to display various components.
- Tailwind CSS Dark Theme: Configure Tailwind CSS with a dark theme by default, featuring a very dark left sidebar and a lighter content area with plenty of whitespace.
- Shadcn UI Component Set: Install and configure shadcn UI with a minimal component set, including button, card, input, label, avatar, badge, dropdown-menu, sheet, skeleton, tabs, and dialog components.

## Style Guidelines:

- Primary accent (buttons, highlights, links): a bright blue around #448BFB.
- Primary hover shade: a slightly lighter blue around #6899FA.
- Dark navy for some headers/strong accents: around #214075.
- Muted slate for secondary UI (borders, side elements): around #4A5C7C.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for headlines or body text.
- Layout: Dark left sidebar and a lighter content area to align with the Pentest-Tools dashboard aesthetic.
